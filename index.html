Değişken Tanımlama

Değişkenlerin başına $ koyarak istediğiniz değeri atayabilirsiniz.

Örnek

$body_color: white;

Elementler Arası SASS İşlemleri

Örnek

div{ h2{ font-family: $font_name; font-size: $font_size; $font_color: gray; color: $font_color } }

Başka bir scss dosyasını çağırma

Örnek

@import "reset";

mixin ve include Kullanımı

mixin ile bir stil tanımı yapıp istenirse başka bir element, class veya id için include ile çağırılarak stil tanımlaması yapılabilir.

Örnek

@mixin fonts{ font-size:20px; font-weight: normal; } p { @include fonts(); }

mixin tanımı yaparken parametreli olacak şekilde de yapabilirsiniz. @include margin(10); olarak çağırdığınızda ilgili yerde "margin:10;" stilini uygulayacaktır.

Örnek

@mixin margin($margin:0){ margin:$margin; }

extend Kullanımı

Tanımlanmış olan stili başka bir stil içinde çağırabilsiniz.

@extend .btn; //şeklinde stil içinde stil çağırabilirsiniz.

Döngüler

Scss sayesinde for döngülerini de kullanabiliriz. Aşağıdaki kullanım ile ".font-size-1" den başlayıp ".font-size-10" a kadar css oluşturacaktır.

@for $x from 1 through 10{ .font-size-#{$x}{ font-size: 0px + $x; } }

Döndü ile oluşan class ı kullandık
